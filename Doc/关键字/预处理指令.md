**预处理指令**

- `__FILE__` 当前文件路径
- `__FILE_NAME__` 当前文件名
- `__DATE__` 当前编译文件日期
- `__TIME__` 当前编译文件时间
- `__LINE__` 当前行号


## `#include_asm` #包含汇编 *.asm 文件到目标位置

```
#import_asm <file>
```

## `#import_sdp` #导入高级宏 *.sdp 文件

```
#import_sdp <file>
```

## `#define` #宏定义

```
#define <name> <value>

#define <name> <value>\
<value>
```

## `#undef` #解除宏定义

```
#undef <name>
```

## `#ifdef` #如果 宏 为真,那么下面的代码会参与编译

```
#ifdef <name>
```

## `#ifndef` #如果 宏 为假,那么下面的代码会参与编译

```
#ifndef <name>
```

## `#endif` #结束条件

```
#endif
```

## `#error` #生成一个编译错误消息

```
#error <string>
```

## `#message` #生成一个编译消息

```
#message <string>
```
